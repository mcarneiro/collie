[WIP] Collie - column module (IE8+ version)

The idea of this module is to create a clean base box modelling of columns with percentage and fixed gutter for you. No setup like column number or column width is needed globally.

This was built for projects that doesn't need (or have) a gridsystem structure.

You'll see some extra elements, such as .col-w that's is needed because of IE8. This is the "old school" way of box modelling. I'll create another module focusing in IE9+ and the markup will get a bit cleaner, also with display:flex options.

Notice that .col-w would just get style of size and margin (top/bottom) and hardly you'd place a background there. On the examples below, consider that the holder of your module is the ._proto-box element. .row and .col-w are just wrappers.

The inline-block technique is good for multiple lines like product lists. By default it has a margin-top: $gutter and a vertical-align: top. It uses the negative margin of $gutter on the holder .row. It also apply a font-size: 0; on .row element to avoid the space between elements.

Responsive behaviour

The responsive behaviour, breakpoints and when to use them is up to you. You can use min-height to apply the modules and have all col-w elements stacked in mobile. If you are working with max-width you can use the row-reset() function, that will make the elements blocked and full with a margin-top of $gutter.
